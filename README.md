## Project Overview

**Phisher Detector** is a simple yet powerful web application that helps users verify the safety of URLs and emails. It flags malicious links, newly registered domains, or insecure sites that could be part of phishing scams.

The project demonstrates practical cybersecurity awareness through front-end web development â€” blending design with functionality.
